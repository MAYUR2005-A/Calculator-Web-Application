# 🧮 Calculator Web App

A simple calculator built using **HTML, CSS, and JavaScript**.
It performs basic arithmetic operations with a clean and easy-to-use interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear (AC) button
* ⌫ Delete (Backspace) functionality
* 🎯 Simple and clean UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 💡 Description

This project is a basic calculator that allows users to perform standard arithmetic operations.
It is designed with a simple interface and is mainly optimized for desktop use.

> ⚠️ Note: This project is **not fully responsive** and may not display properly on smaller screens.
